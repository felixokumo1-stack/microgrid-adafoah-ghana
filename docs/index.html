06 / Key Engineering Insights

What This Project Taught

Five findings with implications beyond this specific project.

Insight 01 — Turbine-site class matching is critical
Vestas V80 (IEC Class I, rated 16 m/s) produced 4.2% CF vs 19.7% for IEC Class III (rated 11 m/s) — a 4.7× gap that would have made wind appear unviable. Turbine class selection must precede simulation.
Insight 02 — Wind integration creates system-level value, not just generation
One EUR 75,000 turbine allowed EUR 95,000 in PV and battery removal, reducing total CAPEX by EUR 23,500 despite adding a component. LCOE improved 21%, NPC by EUR 148,158.
Insight 03 — Diminishing returns are quantifiable before building
Config C's second turbine increased NPC by EUR 14,685. Only 2,968 L/yr diesel remained to displace — turbine lifecycle cost (EUR 104k) exceeded fuel displacement value (EUR 25k).
Insight 04 — Fuel price insensitivity is a design achievement
±40% diesel price swing caused only ±3.5% LCOE change at 97.6% renewable fraction. The system is structurally protected from fuel price volatility.
Insight 05 — IEA MTF thresholds ≠ design consumption values
Using MTF minimum thresholds (200/1,000 Wh/day) produced a 4× load underestimate. GOGLA 2022 typical values (500/1,500 Wh/day) are the correct design basis.
