05 // Optimisation Results — Three Configurations

HOMER Pro evaluated 500–1,730 simulations per configuration. All three met the LPSP ≤ 5% constraint with zero unmet load. Config B (PV + Wind + Battery + Diesel) emerged as the economic optimum at EUR 0.227/kWh — 21% below the solar-only baseline and 63% below diesel-only generation. Adding one 50 kW IEC Class III wind turbine allowed simultaneous PV downsizing by 50 kWp and battery downsizing by 100 kWh, reducing CAPEX by EUR 23,500 despite the turbine addition. Config C confirmed diminishing returns: a second turbine increases NPC by EUR 14,685 because the system is already 97.6% renewable with only 2,968 L/yr of diesel remaining to displace.

06 // Financial Analysis

Against a diesel-only baseline (EUR 0.48/kWh), Config B delivers EUR 89,849/year in cost savings with a 4.7-year simple payback and EUR 882,154 NPV over 20 years. Sensitivity analysis identified discount rate as the dominant uncertainty: LCOE ranges from EUR 0.206/kWh at 6% (concessional) to EUR 0.323/kWh at 15% (commercial). The project break-even financing rate is 16.8% — providing robust headroom above all realistic donor and commercial scenarios. Fuel price sensitivity is minimal (±EUR 0.008/kWh across EUR 0.90–2.00/L range) because Config B consumes only 2,968 L/yr at 97.6% renewable fraction.

07 // Lessons Learned
L1 Turbine-site class matching is critical. Vestas V80 (IEC I, rated 16 m/s) produced 4.2% CF at 6.15 m/s vs 19.7% for IEC Class III (rated 11 m/s). Wrong class invalidates wind analysis entirely.
L2 Cross-validate reanalysis datasets. ERA5 and MERRA-2 disagreed 32% on wind speed at 10 m. Single-source wind data is insufficient for feasibility studies in West Africa.
L3 MTF thresholds ≠ typical consumption. IEA Tier 2/3 minimums (200/1000 Wh/day) underestimated load 4×. GOGLA 2022 typical values (500/1500 Wh/day) are the correct design basis.
L4 Search space boundary artefacts. Initial PV ceiling of 200 kWp was a boundary optimum. Always extend search space beyond first result to confirm true interior optimum.
L5 Rule-based vs optimising dispatch. Python model overstated diesel runtime 49% vs HOMER due to minimum-load-ratio mechanics. LPSP matched exactly. Document discrepancies — do not hide them.
